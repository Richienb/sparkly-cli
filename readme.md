## Install

```
$ npm install --global sparkly-cli
```


## Usage

```
$ sparkly --help

  Usage
    $ sparkly <number> [...]

  Options
    --min    Minimum range
    --max    Maximum range
    --style  Style for the sparklines [Values: fire]

  Examples
    $ sparkly 0 3 5 8 4 3 4 10
    ▁▃▄▅▃▃▃▆
    $ sparkly --min=0 --max=10 1 2 3 4 5
    ▁▂▃▄▄
    $ sparkly --style=fire 1 2 3 4 5 6 7 8
```
